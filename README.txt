Student_Verification_System

Student_Verification_System is a system, which helpfull to student who are plan to get internships, permenent occupations and etc.
Any student who is studying at University of  Jaffna can have a user account in this system. 
If someone have an account in this system he/she can simply put a link of the user account to linkdlin or Curriculum Vitae and then anyone can verify the identity of the particular person.

Here we use,
Vue.js and Laravel as Frameworks
javascript, and  php as Languages
windows as OS
