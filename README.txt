Job_Application_Verification_System

Job_Application_Verification_System is a system, which helpfull to the student who wishes to get internships or permanent occupations.
Any student who is studying at the University of Jaffna can have a user account in this system and his details verified by university. 
If someone has an account in this system he/she can simply put a link of the user account of this system to their LinkedIn or Resume and then anyone can verify the correctness of details of the particular person.

Here we use,
Vue.js and Laravel as Frameworks
javascript, PHP, css, html as Languages
